✋ 21-Landmark Hand Model
MediaPipe outputs these 21 keypoints per detected hand

MediaPipe outputs 21 landmark coordinates per hand in normalised image space (0–1). The game maps them to screen pixels using a safe-zone remap that ignores the outer 15% of camera frame.

● Wrist (0) ● MCP joints (base) ● PIP/DIP joints ● Fingertips

Pinch detection uses the normalised distance between landmark 4 (thumb tip) and landmark 8 (index tip), divided by hand size (wrist→middle MCP distance). Threshold: < 0.28 = pinching.
